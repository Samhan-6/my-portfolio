Calorie Burn Prediction

The CalorieWisePro Web Application is a machine learning-powered tool that helps users to estimate their calorie expenditure based on their unique characteristics. This application, which was created as a part of a B.Sc. in Software Engineer final project, makes use of the features of the XGBoost machine learning library and the well-known Python web framework Django. Accurate calorie burn projections may be obtained by users by entering their gender, age, height, weight, workout time, heart rate, and body temperature.
